  2. 创建新的仓库：

    • 登录GitHub账号。
    • 在GitHub页面的右上角，点击 “+” 图标，然后选择 “New repository”。
    • 填写仓库名称（例如：LeetCodeSolutions），可以选择公开或私有仓库。
    • 可以选择初始化仓库，比如添加README文件。
    • 点击 “Create repository”。
